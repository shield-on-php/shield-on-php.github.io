setRdnsList

  • param array $data Aarry collects RDNS (IP's resolved hostname) strings.
  • param string $type type [allow, deny, searchbot]
  • return void
$list = [
    '.googlebot.com',
    '.google.com',
    '.live.com',   // search.live.com
    '.msn.com',    // msnbot.msn.com, search.msn.com
    '.ask.com',
    '.bing.com',
    '.inktomisearch.com',
    '.yahoo.com',
    '.yahoo.net',  // crawl.yahoo.net
    '.yandex.com',
    '.yandex.ru',
];

$robot->setRdnsList($list, 'allow');

  • param string $type type [allow, deny, searchbot]
